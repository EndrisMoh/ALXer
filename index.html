About

ALXer was concieved when I was challenged to configure NGINX server at the ALX_Africa Software Engineering program. Then when I was given assignment to research for topics for portfolio project I asked my self, why don't I try to develp a simple web server that can be configured and used easily. Then I tried to research how can I develop this simple web sever using Python programming language. I got some resouces about Network programming, sockets and TCP/IP models. Then I decided to develop it and that gives the birth of ALXer. This portfolio project took me more than a month for project research and approval, MVP specification, MVP development, testing and deployement phases.
